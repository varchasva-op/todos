# 📝 Todo REST API

A clean, RESTful API for managing todos, built with **Node.js** and Express.js. Supports full CRUD operations with proper HTTP status codes, input validation, and structured JSON responses.


🚀 Features

- ✅ **Create** new todos with title & description
- 📋 **Read** all todos or fetch a single todo by ID
- ✏️ **Update** todo details or mark as completed
- 🗑️ **Delete** todos by ID
- 🛡️ Input validation with meaningful error messages
- 📦 Structured JSON responses
- ⚡ Fast in-memory storage (no database setup required)



 🛠️ Tech Stack

- **Runtime:** Node.js
- **Framework:** Express.js
- **Dev Tools:** Nodemon



📦 Installation

 Prerequisites
- Node.js (v14 or higher)
- npm or yarn
